# 🍕 ДУДУ Пиццерия

Веб-приложение для онлайн-заказа пиццы с админ-панелью, табло готовых заказов и загрузкой изображений. Учебный проект, демонстрирующий полный цикл разработки: от авторизации до управления заказами.

## ✨ Функционал

**Для пользователя:**
- Регистрация и вход (с валидацией email и пароля)
- Просмотр меню с пиццами (название, цена, ингредиенты, фото)
- Добавление и удаление пицц из корзины
- Оформление заказа
- История своих заказов с отслеживанием статуса

**Для администратора:**
- Вход по отдельным учётным данным
- Управление статусами заказов: «Ожидает» → «Готовится» → «Готово» → «Выдан»
- Добавление новых пицц в меню (название, цена, ингредиенты, загрузка фото)
- Просмотр всех заказов в админ-панели

**Общее:**
- Табло готовых заказов — автоматическое отображение заказов со статусом «Готово»
- Загрузка файлов — предпросмотр, проверка типа (JPEG, PNG, WEBP) и размера (до 2 МБ)
- Адаптивная вёрстка — CSS Modules, сетка, анимации

---

## 🛠 Технологический стек

- **Фронтенд:** React 19, TypeScript, Vite
- **Стилизация:** CSS Modules, глобальные стили
- **Управление состоянием:** Zustand
- **Маршрутизация:** React Router v6
- **Бекенд (моковый):** JSON Server
- **Загрузка файлов:** Моковая (public/uploads)
- **Валидация форм:** Кастомные утилиты

## 🧪 Локальный запуск

**Требования:** Node.js (версия 18 или выше), npm

**Установка и запуск:**

```bash
# 1. Клонировать репозиторий
git clone https://github.com/reinkarnatsiya/dydy-pizza.git
cd dudu-pizzeria

# 2. Установить зависимости
npm install

# 3. Запустить JSON Server (бекенд) в первом терминале
npm run server

# 4. Запустить React-приложение во втором терминале
npm run dev
```

 Проект создан в учебных целях. Не для коммерческого использования.
